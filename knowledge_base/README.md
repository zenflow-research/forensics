## Quick Start

### For Manual Company Analysis
1. Start with `05_CHECKLIST.md` -- follow the 18-step protocol
2. Reference `02_RED_FLAGS_TAXONOMY.md` as you analyze
3. Cross-check findings against `07_MANIPULATION_TECHNIQUES.md`
4. Compare with similar companies in `04_COMPANY_CASE_STUDIES.md`

### For AI-Powered Analysis
1. Use prompts from `06_PROMPT_TEMPLATES.md` with any LLM
2. For building a local system, follow `08_LLM_TRAINING_GUIDE.md`
3. Start with RAG approach (fastest to deploy)

### For Learning
1. Read `01_FRAMEWORK.md` for the complete methodology
2. Study `10_KEY_CONCEPTS_GLOSSARY.md` for terminology
3. Review case studies in `04_COMPANY_CASE_STUDIES.md`
